## 加载动画

长图制作逐帧动画

### 基本使用

:::demo 基本使用方式。

```js
render(){
  return(
    <Loading/>
  )
}
```
:::


### Loading
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| loadingIcon  | 加载中动画图    | object   | — | —   |
| extraCls  | 额外的class    | string   | — | —   |
