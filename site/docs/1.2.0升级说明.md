## ppfish 1.2.0 版本升级说明

#### 1.ReactAmap
 
 - ReactAmap组件从ppfish中删除，迁移至prophet业务组件目录。
 
#### 2. Echarts 

 - Echarts组件不从index暴露，但保留源码。
 
### 3. DayTimeSelect

 - DayTimeSelect组件不从index暴露，但保留源码。
 
### 4. TimePicker

 - TimePicker更名为BizTimePicker
 - active属性更名为visible （用于手动控制浮层显隐）
 - 增加属性clickAreaStyle，设置点击区域的样式
